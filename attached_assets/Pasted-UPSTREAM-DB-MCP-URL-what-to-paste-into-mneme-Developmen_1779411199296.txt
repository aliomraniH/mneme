Tools mneme can call against this MCP server
Tool	Purpose

get_artist	one artist with links / images / provenance

Safety guarantees baked in

    The MCP server only ever touches the 6 saaz tables (SAAZ_TABLES whitelist for describe_table).
    Every query call is wrapped in SET LOCAL default_transaction_read_only = on + statement_timeout = 10s and pre-validated to reject anything that isn't a single SELECT/WITH-SELECT.
    mneme's tables (query_episode, expertise_note, etc.) are not reachable through this server at all.
